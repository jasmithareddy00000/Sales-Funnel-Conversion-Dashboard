**Sales Funnel Conversion Dashboard**
# Tools Used
- MySQL
- Power BI

# Overview
This project analyzes lead conversion across sales funnel stages using MySQL and Power BI.

# Features
- Funnel visualization (Leads → Opportunities → Wins)
- Monthly trend analysis
- Region-wise performance
- Interactive slicers for region and date

# Files
- SQL scripts for data extraction
- Dashboard screenshots
- Powerbi conversion Dashboard
